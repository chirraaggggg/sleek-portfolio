# Sleek Portfolio 

A modern, responsive developer portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **shadcn/ui**, featuring projects, blog posts, experience timeline, and a contact form â€“ all optimized for performance, accessibility, and SEO. 

## Features 

- Next.js App Router with modern file-based routing.   
- TypeScript-first codebase for better DX and safety.   
- Tailwind CSS styling with shadcn/ui components.  
- Responsive layout with dark/light mode toggle.   
- MDX-powered blog and project pages.   
- SEO-friendly meta tags and structured content.   
- Contact form with Telegram integration (configurable).   
- Analytics integration support (e.g. Umami) for privacy-friendly tracking.   

## Tech Stack 

- **Framework:** Next.js 15 (App Router) 
- **Language:** TypeScript 
- **Styling:** Tailwind CSS, shadcn/ui   
- **Content:** MDX for blog and projects   

## Overview 

This portfolio is designed to be clean, fast, and easy to customize for developers who want to showcase their work, experience, and writing in a sleek interface. 
Update the configuration and content files to plug in your own details, projects, and posts, and you have a production-ready personal site in minutes.
